## 简介

LLM-ROUTE 是一个轻量级 LLM API 路由工具，主要解决大模型厂商超售导致请求频繁超时或出错的问题。

## 主要功能

- **反向代理** - 支持 Anthropic 和 OpenAI 协议
- **智能重试** - 自动重试失败的请求
- **Responses API 转换** - 兼容更多模型服务
- **系统托盘** - 便捷的图形界面
